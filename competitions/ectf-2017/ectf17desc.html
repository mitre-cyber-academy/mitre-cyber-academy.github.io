Collegiate eCTF 2017

Overview

MITRE's 2nd collegiate eCTF (embedded capture-the-flag) is an embedded security competition that puts participants through the experience of trying to create a secure system and then learning from their mistakes. The main target is a real physical embedded device, which opens the scope of the challenge to include physical/proximal access attacks. As a pilot competition, participation in this event is invitation only. If you're interested in participating in eCTF events, please contact us at ectf@mitre.org
